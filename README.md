# Haki Legal Chatbot 2026

An AI-powered (rule-based) assistant designed to provide basic information regarding Kenyan Labor Laws and Tenant Rights.

## Features
- **Labor Law Guidance**: Information on leave days, termination procedures, and minimum wage.
- **Tenant Rights**: Information on eviction processes, rent deposits, and rent increases.
- **Swahili/English Context**: Recognizes key legal terms used in the Kenyan context.

## Usage
Run the chatbot using Python:
```bash
python chat.py
```

## Legal Disclaimer
This tool provides general information based on the Employment Act and the Landlord and Tenant Act of Kenya. It does not constitute formal legal advice. For specific cases, users should consult a certified legal practitioner or the Law Society of Kenya (LSK).
